# Background Image Setup

Please download a suitable fisheye town background image and save it as `town-bg.jpg` in this directory.

Recommended specifications:
- Resolution: At least 1920x1080
- Format: JPG
- Style: Fisheye view of a city/town
- Content: Urban landscape with buildings and streets
- Tone: Light/neutral to ensure text readability when used as background

You can find suitable images from:
1. Unsplash.com
2. Pexels.com
3. Adobe Stock
4. Shutterstock

After downloading, rename the image to `town-bg.jpg` and place it in this directory.
