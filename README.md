# NEEDS Platform

A Supabase-first marketplace application with Next.js frontend and Stripe Connect payments.

- **[Architecture](docs/ARCHITECTURE.md)**: System overview, tech stack, and data models.

- **[Security Audit](docs/SECURITY_AUDIT.md)**: Security findings and recommendations.

## Project Structure

- `frontend/`: Next.js application and Supabase configuration.
- `docs/`: Project documentation.

## Quick Start

```bash
# Clone the repo
git clone <repository-url>

# Install dependencies
cd needs-root/frontend
pnpm install

# Run dev server
pnpm dev
```

See [SETUP.md](docs/SETUP.md) for detailed instructions.
